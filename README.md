# UK Crime

A webpage to show last year's most common crime at any UK postcode.

[Live demo](http://ukcrime.net/)

## APIs used

* [postcodes.io](http://postcodes.io/) - For UK postcode lat/lng
* [data.police.uk](https://data.police.uk/) - For UK crime data
